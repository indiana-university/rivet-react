### Example
```jsx
<Dropdown label="Navigation menu">
    <a href="#">Item one</a>
    <a href="#">Item two</a>
    <a href="#" aria-current="page">Item three</a>
    <a href="#">Item four</a>
    <DropdownGroup>
        <a href="#">Related item three</a>
        <a href="#">Related item four</a>
    </DropdownGroup>
</Dropdown>
```

### Right-align dropdown
```jsx
<Dropdown label="Right menu" align="right" variant="secondary">
    <a href="#">Item one</a>
    <a href="#">Item two</a>
    <a href="#" aria-current="page">Item three</a>
    <a href="#">Item four</a>
</Dropdown>
```
